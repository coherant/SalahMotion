# Prayer Library

Centralised list of all prayers used across positions in the state machine.
Reference a prayer in any position's `utterance` cell using its `prayer-id` (e.g. `P-1`).
The same `prayer-id` can appear in multiple positions without duplicating the text.

This file is shared by all prayer profiles (calibration, guided, etc.).
